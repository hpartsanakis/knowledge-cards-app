# Knowledge Cards App

A small Progressive Web App (PWA) to manage personal knowledge cards.

The app allows you to create, edit, search, sort and organize cards into different collections such as photography moments, recipes and travel destinations.

The application works offline and can be installed like a mobile app.

---

## Features

- Create, edit and delete cards (CRUD)
- Accordion card layout
- Collections system
- Live search
- Sorting (A–Z, Z–A, newest, category)
- Image upload for photo cards
- Local storage persistence
- Installable Progressive Web App (PWA)
- Offline support via Service Worker

---

## Collections

The app includes three main collections:

📸 Photo Moments  
Store photography notes and images.

🍳 Recipes  
Save your favorite recipes and cooking ideas.

✈️ Travel Destinations  
Keep notes about places you want to visit.
